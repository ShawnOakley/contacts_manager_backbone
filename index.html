A contacts manager app using Backbone.js. Once can add contacts, sort by any of the fields, edit any of the contacts, and add categories for each contact.

Backbone Contacts Manager

This app uses Backbone.js to make the front-end of a contacts manager. It's currently not connected to a database, meaning that the contacts can't be persisted. The user can add contacts, delete, sort, edit, and add categories for contacts.
